A bit more serious attempt to write an app to click on any point on the screen with just keyboard. Meant to work in tandem with my 34-key split keyboards, thus not meant to be anything else but part of my personal setup.

Additionally it also does language switching because Windows 11 introduced bugs with it's new implementation, see: https://techcommunity.microsoft.com/discussions/windows11/windows-11-language-switch-hot-keys-dont-always-work-or-work-slowly-/4105408/replies/4417535
